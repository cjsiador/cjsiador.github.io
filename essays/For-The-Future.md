   When I was in High School, my STEM teacher emphasized how important Computer Science is. He taught us some information about algorithms, cyber security, and how to program HTML - CSS - JavaScript such as making headers and for loop. One day, he gave us an assignment to create a webpage using the three programming languages. I became so obsessed with creating my very own HTML website that I decided to do more research on what I can do for programming my website. It had CSS animations and video/audio played on my website. It made me realize that creating new things through the computer is something that made me very passionate about programming. I guess if it wasn’t for my STEM teacher, I would have graduated with a Liberal Arts degree instead of Computer Science. 

   It made me realize that Software Engineering is something that interests me. What’s fascinating about computers and technology is that it continues to evolve to help shape our society. Yet, I want to be that person who contributes for the future. Now that I’m taking a Software Engineering course, I am understanding more about Software Engineering and it peaked my interest. I am interested in creating projects that have a meaningful impact on others and learn more about becoming a Software Engineer. Also, I like to stay up all night writing codes.

   Hopefully, in the future I can develop skills and experience to contribute to society. That’s why I have some goals in mind that I would like to achieve. I’d like to learn how to create machine learning algorithms and implement them to real life problems. However, this isn’t easy and it’ll take some time learning this field. Another goal of mine is to improve on communication so that whenever there is a group project, I’ll be able to be more fluent and organize with my fellow members. Lastly, I want to improve my analytical skills and improve my capabilities of solving problems within a given amount of time. This way, I’ll be ready for a coding interview in the future and become someone who helps make the future.
