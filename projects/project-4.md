Our journey has come to an end as a new chapter ascends. From here on out, I won’t have any doubts. But, when time has come, I’ll give it a shout. Throughout 2020, it was a very difficult time of year, but it’ll never stop me from learning new things. I realized that online format gave me the ability to be more productive and time to study, however, there were some challenges I’ve faced out of fear of the outdoors and the feeling of isolation. Though, I was able to combat the challenge of isolation with the help of connecting with other students as a community and work in a team of three. 

I learned that working in a team requires strong communication and understanding of each other. Instead of meeting in person, the most optimal solution was to utilize other communication applications such as Discord and most importantly Github. Discord was used for meetings and communication about the project itself. However, it was also used as a place to feel connected and share stories. Such in a way to learn about each other that creates a bond between the team members. As for Github, it was an environment for our project whereas the “Master” branch was used for combining our programs together with the respected responsibility of one's' teammate duty based on our project board, or as we called it a milestone. These types of activities represent Agile Project Management where I learned that it’s very important for communication and working as a team. 
 
Communication is not always through verbal, but also the way we code in our program. Coding standards is a concept that I learned about creating a project, that is hopefully create a more understandable code for others to read. Eslint is a tool that helps me to find errors using coding standards and pinpoint unnecessary code that makes it easier to read. It also enables reducing the amount of lines of code. Most importantly, companies such as Facebook or Amazon have rules of how coding should be formatted and this portrays the importance of the concept coding standards. I’ve seen codes that violate coding standards rules and it ended up with such headaches. Imagine with all the unnecessary indentation or tens of lines of code that are empty. It creates such a huge problem and yet, it’s important to implement these coding standards to save time and other’s sanity.
