Manoa Clubs Finder is a website that was developed for other students to search for UH clubs. 
The functionality of Manoa Club Finder is that it utilized the ability for club leaders to edit their own clubs. Additionally, users who are club members have the ability to find and search a club with information about the club itself. Information such as the location, description, name, club’s leader contact info, and the meeting times so that other users could view it as a whole. It also has a built in filter system that filters based on the selected categories with the club’s selected category. The Super Admins’ role is to monitor the website and initiate new clubs that must be added to the database for the club. 

My role for this project is to create a filter system whereas it filters the categories. The website has a schema where it acts almost like a constructor, but intakes new data and assigns new objects. This was used to gather the club’s category variable and implemented into the filter system where it lists all the clubs that met the criteria. Additionally, the club search was designed by me using semantic ui react.

The outcome of this project was not only learning about creating a website, but also working as a team. I learned that it’s important to have a leader, or a project manager within a team. Without it, there could be some loose ends and miscommunication. It’s important to set meetings to discuss problems pertaining to the project, but it’s also important to have a bond between the other members. It was hard for me to communicate where we knew little about each other where it was a bit uncomfortable. However, as time went on, it became easier to be more comfortable where we all talk about stuff not related to the project. 

You can learn more at [manoa-clubs-finder.github.io](https://manoa-clubs-finder.github.io/).
